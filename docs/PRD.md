## 1. 產品概述

### 1.1 產品定位

大六壬 Zenith 是一個專業級的大六壬占卜排盤系統，融合傳統術數精髓與現代軟件工程，提供準確、詳盡、易用的占卜推演服務。

### 1.2 核心價值

- **準確性** - 精確的曆法計算與嚴謹的排盤邏輯
- **詳盡性** - 四課三傳、課體判斷、用神分析全面覆蓋
- **易用性** - 現代化界面設計，REST/gRPC 雙接口支持

### 1.3 目標用戶

- 大六壬學習者與研究者
- 專業占卜師
- 傳統文化愛好者
- 需要 API 集成的開發者

- 遙克法、昴星法、別責法、八專法

## 4. 算法設計

### 4.1 十神計算算法

```go
// 判斷邏輯
1. 同我者：比肩（同陰陽）/劫財（異陰陽）
2. 我生者：食神（同陰陽）/傷官（異陰陽）
3. 我剋者：偏財（異陰陽）/正財（同陰陽）
4. 剋我者：七殺（同陰陽）/正官（異陰陽）
5. 生我者：偏印（異陰陽）/正印（同陰陽）

// 陰陽判斷
- 天干：甲丙戊庚壬為陽，乙丁己辛癸為陰
- 地支：子寅辰午申戌為陽，丑卯巳未酉亥為陰
```

### 4.2 智能建議算法

#### 天將分類
- **吉神**: 貴人、青龍、六合、太常
- **凶神**: 螣蛇、朱雀、白虎、玄武、天空
- **中神**: 勾陳、天后

#### 十神分類
- **幫扶類**: 比肩、劫財、正印、偏印
- **洩耗類**: 食神、傷官
- **克制類**: 正官、七殺
- **財利類**: 正財、偏財

#### 組合邏輯
| 天將 | 十神 | 建議方向 |
|-----|------|---------|
| 吉神 + 幫扶 | 順勢而為，積極進取 |
| 吉神 + 財利 | 求財有利，合作順遂 |
| 吉神 + 克制 | 有貴人解厄，宜謙卑求助 |
| 凶神 + 克制 | 壓力重重，靜守避禍 |
| 凶神 + 幫扶 | 有助力可解凶 |
| 凶神 + 洩耗 | 防範暗算，謹守財物 |

### 4.3 階段性判斷

#### 初傳（發端）
- 正官/正印 → 最佳時機，宜立即行動
- 七殺/偏印 → 需謹慎，充分準備
- 正財/偏財 → 有利可圖，把握機會

#### 中傳（過程）
- 正官/正印 → 過程順遂，按部就班
- 七殺/偏印 → 過程有阻，需調整策略

#### 末傳（結果）
- 吉神 + 吉十神 → 結果圓滿
- 凶神 + 凶十神 → 需防損失

## 5. API 規格

### 5.1 REST API

#### 排盤接口
```
POST /api/v1/divination
Content-Type: application/json

Request:
{
  "date": "2026-03-16",      // 日期（必填）
  "time": "14:00",           // 時間（必填）
  "question": "問財運",       // 占事（可選）
  "question_type": "財運"     // 問題類型（可選）
}

Response:
{
  "success": true,
  "data": {
    // 四柱
    "YearPillar": "丙午",
    "MonthPillar": "辛卯",
    "DayPillar": "己丑",
    "HourPillar": "辛未",
    
    // 四課（含十神與意義）
    "FourKe": [
      {
        "Number": 1,
        "Meaning": "占者自身狀態、表象",
        "Down": "丑", "Up": "亥",
        "Relation": "劫財"
      }
    ],
    
    // 三傳（含智能建議）
    "SanChuan": {
      "Chu": { "Branch": "巳", "General": "青龍", "Relation": "正印" },
      "ChuDetail": {
        "Explanation": "【發用·起因】青龍臨巳...",
        "Advice": "【發端】青龍主財祿...",
        "KeyPoints": ["貴人提攜", "利於仕途"]
      }
    }
  }
}
```
